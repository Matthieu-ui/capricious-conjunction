## Project Goals

The goal of this project is to create a community board for developers to stay up to date with the Steam Web Api and create a community of developers that can help each other with issues and ideas. 

## Project Features

The project features a content management system that allows users to create posts and share their projects and ideas. The project also features a search bar that allows users to search for specific games and view the game's data. The project also features a user authentication system that allows users to create accounts and login to the discussion board via OpenID.
