## What it is

The in-game CS2 server browser is slow and awkward — filtering is limited, connecting takes too many clicks, and finding a specific surf server is more friction than it needs to be.

This is a third-party browser built on the Steam Web API that cuts that friction. Filter by map, ping, player count, or server name. Connect directly. Built for the surf community specifically, where you're looking for a particular tier or map and the default tools just get in the way.

## Stack

React frontend, Node backend hitting the Steam Web API, Supabase for persistence, Tailwind for styling.

In progress.
