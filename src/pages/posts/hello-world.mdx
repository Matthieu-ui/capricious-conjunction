## Introduction

This is a simple Twitter API application. It uses the Twitter API to fetch tweets from a user's timeline. It also uses the Twitter API to fetch a user's profile information.

## Code

```js
import { TwitterApi } from "https://deno.land/x/twitter_api/mod.ts";

const twitterApi = new TwitterApi({
  consumerKey: Deno.env.get("TWITTER_CONSUMER_KEY"),
  consumerSecret: Deno.env.get("TWITTER_CONSUMER_SECRET"),
  accessTokenKey: Deno.env.get("TWITTER_ACCESS_TOKEN_KEY"),
  accessTokenSecret: Deno.env.get("TWITTER_ACCESS_TOKEN_SECRET"),
});
```
